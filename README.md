User Management MERN Application

This project is a simple MERN stack application that demonstrates basic CRUD (Create, Read, Update, Delete) operations for managing users. Each user contains a name, email, and age. The application is built to showcase backend API development, frontend integration, and clean state management without unnecessary complexity.

Tech Stack

Frontend

1. React (Functional Components & Hooks)
2. Axios
3. Tailwind CSS

Backend

1. Node.js
2. Express.js
3. MongoDB
4. Mongoose

Features
-> Add a new user with basic validation
-> View all users in a tabular format
-> Update existing user details using a modal-based edit flow
-> Delete users without page reload
-> Email uniqueness validation on the backend
-> Centralized API handling using Axios interceptor
-> Clean UI with seamless state updates
